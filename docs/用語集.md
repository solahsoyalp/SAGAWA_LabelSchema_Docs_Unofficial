## 用語集

本リポジトリのレイアウト／コードで使われる主な用語の早見表です（非公式の整理）。

### 配送サービス

| 用語 | 説明 |
|------|------|
| 飛脚宅配便 | 標準の宅配便サービス。スピード指定 `000`。 |
| 飛脚航空便 | 航空輸送を使う速達サービス。翌日中配達 `003` / 翌日午前中配達 `004`。 |
| 飛脚ジャストタイム便 | 配達時刻を時分単位で指定できるサービス。スピード指定 `005`。 |
| 飛脚クール便 | 冷蔵（`002`）／冷凍（`003`）の温度管理輸送。 |
| eコレクト | 代金引換サービス。現金／デビット・クレジット等の決済種別を指定。 |
| SRC | Sagawa Returns Center 等で用いられる返品関連の区分（`SRC区分`）。 |

### 送り状・伝票

| 用語 | 説明 |
|------|------|
| お問い合せ送り状No. | 荷物を追跡するための送り状番号。省略時はシステムが自動採番。 |
| 元着区分 | 運賃の負担区分。`1`=元払（発送人負担）／`2`=着払（受取人負担）。 |
| 指定シール | 取扱注意・天地無用・時間帯指定などをラベルに印字するための区分。 |
| 営業所受取 | 受取人が営業所で荷物を受け取るサービス。 |
| 荷姿 | 梱包形態（箱類・バッグ類・封筒類など）。 |

### 関係者・コード

| 用語 | 説明 |
|------|------|
| お届け先 | 荷物の届け先（受取人）。 |
| ご依頼主 | 配送を依頼する主体。荷送人と異なる場合がある。 |
| 荷送人 | 実際に荷物を発送する主体（差出人）。 |
| 部署ご担当者 | 荷送人側の部署・担当者情報。 |
| 取得区分 | 住所録／担当者を登録した際のサービス種別の区分（`0`=飛脚宅配便 / `1`=サービス共通 等）。 |
| JIS地区コード | 住所を表す JIS の地区コード（5桁／8桁／11桁）。 |

### 集配状態

| 用語 | 説明 |
|------|------|
| 集配状態コード | 配達・集荷の進捗を表すコード（[集配状態コード一覧](集配状態コード一覧.md)）。 |
| 消込（けしこみ） | 配達完了などを確定処理すること（[貨物消込集配状態コード](貨物消込集配状態コード.md)）。 |
| 配完（はいかん） | 配達完了。ドライバー確認段階の「仮完了」を含む。 |
